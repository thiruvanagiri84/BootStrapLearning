Hiding labels 
            A horizontal form means that the labels are aligned next to the
            input field (horizontal) on large and medium screens.
            On small screens (767px and below), it will transform
            to a vertical form(labels are placed on top of each input).
            Additional rules for a horizontal form:
                Add class .form-horizontal to the 
 element
                add class .control-label to all

Input Groups

The .input-group class is a container to enhance an input by adding an icon, text or a button in front or behind it as a "help text".

The .input-group-addon class attaches an icon or help text next to the input field.
