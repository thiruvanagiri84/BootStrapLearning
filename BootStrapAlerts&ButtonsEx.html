The aria-* attribute and × explanation

To help improve accessibility for people using screen readers, you should include the aria-label="close" attribute, when creating a close button.

    & times; (×) is an HTML entity that is the preferred icon for close buttons, rather than the letter "x".

    Closing Alerts
× Click on the "x" symbol to the right to close me.
To close the alert message, add a .alert-dismissable class to the alert container. Then add class="close" and data-dismiss="alert"
    to a link or a button element (when you click on this the alert box will disappear).

Alert class and Alert Links - Used to show error message to user or any specific message to user

Add the alert-link class to any links inside the alert box to create "matching colored links".
